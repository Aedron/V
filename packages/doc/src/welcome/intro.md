# 介绍

Vize：代化的 Web 可视化编排引擎

Github: [https://github.com/vize-team/vize](https://github.com/vize-team/vize)

![stars] ![forks] ![issues] ![pr] ![closed-pr]

## ✨ 特性

- 丰富的物料抽象粒度
- 强大的可配置事件系统
- 友好、便捷的可视化编辑体验
- 支持物料间联动
- 友好、一致的物料 API
- 完全所见即所得
- 丰富的拓展接口
- 物料与系统完全解耦
- 完备的开发工具链支持
- 开箱即用的接入方式
- ...
